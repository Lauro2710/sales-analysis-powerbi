# 📊 Análisis de Ventas con SQL y Power BI

## Descripción

Este proyecto analiza el rendimiento de ventas utilizando la base de datos de ejemplo **Classic Models**. Mediante consultas SQL y un dashboard interactivo desarrollado en Power BI, se identifican tendencias de ventas, productos con mejor desempeño, vendedores destacados y otros indicadores relevantes para apoyar la toma de decisiones.

## Base de datos

Se utilizó la base de datos **Classic Models**, ampliamente empleada para practicar SQL, análisis de datos y visualización.

Las tablas principales utilizadas fueron:

- Customers
- Orders
- OrderDetails
- Products
- ProductLines
- Employees
- Offices
- Payments

## Metodología

1. Se realizaron consultas SQL en MySQL para responder diferentes preguntas de negocio.
2. Los resultados de cada consulta se exportaron a archivos CSV.
3. Los archivos CSV fueron importados en Power BI para construir un dashboard interactivo.
